# Jeg kan gjøre mange ting

##
Jeg kan lage mat.

##
Jeg kan sykle.

##
Jeg kan sy.

##
Jeg kan jobbe hardt.

##
Jeg kan snakke tre språk.

##
Jeg kan synge.

##
Jeg kan passe barn.

##
Jeg kan le!
