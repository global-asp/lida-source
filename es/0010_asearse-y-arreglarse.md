# Asearse y arreglarse

##
Él se lava las manos.

##
Se lava la cara.

##
Se cepilla los dientes.

##
Se afeita.

##
Se ducha.

##
Se lava el cabello.

##
Se seca.

##
Se peina el cabello.

##
Se corta las uñas.
